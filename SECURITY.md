# Security Policy

## Supported versions

Security fixes are provided for the latest published version of Azoth.

## Reporting a vulnerability

Please report suspected vulnerabilities privately by opening a GitHub security advisory for this repository.

Do not disclose security issues publicly until maintainers have investigated and coordinated a fix. Include reproduction steps, affected versions, and any relevant logs or proof-of-concept details.

Maintainers will acknowledge reports as soon as practical and will coordinate remediation through a private advisory before public disclosure.
