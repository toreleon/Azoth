# Azoth v0.0.2

Release date: 2026-05-04

Azoth v0.0.2 is a packaging and release-readiness update for the first public npm distribution. It keeps the runtime surface unchanged while tightening CI behavior and npm package metadata.

## Changes Since v0.0.1

- Enabled pnpm native dependency builds for `better-sqlite3`, so GitHub Actions installs the SQLite binding required by runtime and test workflows.
- Cleaned npm `bin` metadata for the packaged CLI commands so npm preserves the executable entries during publish validation.
- Removed outdated TradingAgents gap-analysis/checklist/internal documentation that no longer reflects the current Azoth release baseline.
- Bumped the package version from `0.0.1` to `0.0.2`.

## Package Commands

The npm package continues to expose:

- `azoth`
- `azoth-init`
- `azoth-analyze`
- `azoth-team`
- `azoth-backtest`
