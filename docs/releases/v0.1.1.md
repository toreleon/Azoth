# Azoth v0.1.1

Release date: 2026-05-04

Azoth v0.1.1 republishes the v0.1 public baseline through the GitHub Actions
release pipeline after the locally published v0.1.0 artifact was unpublished.
There are no runtime feature changes from the v0.1.0 baseline.

## Changes

- Publish through the protected-branch GitHub release workflow.
- Preserve the single public `azoth` CLI command.
- Keep the v0.1 agent team, TUI, backtest, roadmap, and release documentation
  baseline intact.
